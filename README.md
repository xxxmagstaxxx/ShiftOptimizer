サークル (東大装い喫茶研究会) 用。
Googleフォームの内容からシフト希望を取り出し、そのシフト希望から最適なシフトを作るプログラムです。

手順

1. shift_query.csvにGoogleフォームの内容を正しいフォーマットになるようにコピー
2. shift_convert.pyを実行 (Googleフォームの内容からシフト希望を取り出すプログラム)
3. shift_shaped.csvが生成され、場合に応じて手直し
4. shift_optimize.pyを実行 (シフト希望から最適なシフトを作るプログラム)
5. shift_best.csvが生成され、最適なシフトが出力されている
